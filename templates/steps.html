Analysis Pipeline

The execution lifecycle of an uploaded image sample

1. Normalization:

Image dimensions are resized to uniform specifications, and color pixel values are rescaled (typically to a 0–1 range) to ensure compatibility with deep learning input tensors.

2. Feature Extraction:

Deep convolutional layers traverse the image to identify and map low-level textures, edge orientations, and contrast variations, effectively creating a hierarchical representation of the visual data.

3. Classification:

The system maps the extracted feature vectors into a learned probability space. It then evaluates these against trained models to determine the likelihood of human-created patterns versus synthetic artifacts produced by diffusion generation models.
